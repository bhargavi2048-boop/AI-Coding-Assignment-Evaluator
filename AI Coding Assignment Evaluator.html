          üìò About This Project

          This project was built for the AI Coding Assignment Evaluator Hackathon. It addresses key challenges in coding assessment such as delayed feedback, surface-level scoring, lack of explainability, scalability issues, and unfair evaluation. Our mission is to design an AI-powered evaluator that automatically assesses correctness, efficiency, readability, modularity, edge cases, and best practices. It generates clear, constructive feedback and ensures fair, consistent evaluation at scale. Deliverables include a working prototype, evaluation metrics, a feedback engine, plagiarism detection, and a demo presentation. Success means a student submits code and walks away knowing correctness, efficiency, strengths, weaknesses, and how close they are to industry standards. Evaluation criteria include impact, innovation, technical execution, user experience, and presentation.
